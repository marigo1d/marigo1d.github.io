核心功能

  1. 项目初始化

创建文件夹 .taskmaster 和 .xxx(对应IDE rules)

.taskmaster

  • config.json 配置各模式使用LLM
  • templates PRD文档模板
  • tasks 包含文件 tasks.json PRD文档解析后得到的任务列表
  • docs 用户自行创建文件 prd.txt

.xxx(对应IDE rules)

  • rules IDE基本规则，参考cursor rules, trae rules 和 rules编写
  • mcp.json 配置LLM API key
