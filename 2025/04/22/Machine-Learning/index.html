机器学习基本概念

监督学习

分类算法（Classification Algorithms）

  • 任务目标：预测一个离散的类别标签（如是否患病、邮件是否是垃圾邮件）
  • 常见算法包括：
    • 逻辑回归（Logistic Regression）
    • K近邻（K-Nearest Neighbors）
    • 决策树（Decision Tree）
    • 随机森林（Random Forest）
    • 支持向量机（SVM）
    • 梯度提升树（XGBoost, LightGBM）
    • 神经网络（用于图像分类、文本分类等）

回归算法（Regression Algorithms）

  • 任务目标：预测一个连续值（如房价、股票价格、温度）
  • 常见算法包括：
    • 线性回归（Linear Regression）
    • 岭回归 / 套索回归（Ridge / Lasso）
    • 决策树回归（Decision Tree Regression）
    • 随机森林回归（Random Forest Regressor）
    • 支持向量回归（SVR）
    • 神经网络（用于非线性回归任务）

非监督学习

  • 无标签数据，目标是发现数据的结构、规律、分布

  • 常见算法包括：

    聚类（Clustering）

    • K-Means, DBSCAN, 层次聚类
    • 例子：客户画像、文档聚类

    降维（Dimensionality Reduction）

    • PCA, t-SNE, UMAP
    • 例子：可视化高维数据、数据压缩

    异常检测（Anomaly Detection）

    • Isolation Forest, 一类SVM
    • 例子：信用卡欺诈检测、工业设备故障

强化学习

Q-Learning, SARSA（基于值）

Policy Gradient, Actor-Critic（基于策略）


训练、验证、测试集


过拟合与欠拟合


模型评估指标

精度、召回率、F1、AUC等



常用算法

监督学习

线性回归

回归

通过 （Feature，Label）训练函数，通过特征输入预测标签

一般的，线性回归模型的方程为
$$
y’ = b + w_1x_1
$$
其中：

  • y′ 是预测的标签，即输出。
  • b 是模型的偏差。偏差与线性代数方程中的 y 截距的概念相同。在机器学习中，偏差有时称为 w0。偏差是模型的参数，在训练期间计算得出。
  • w1 是特征的权重。权重与线性代数方程中的斜率 m 的概念相同。权重是模型的参数，在训练期间计算得出。
  • x1 是特征，即输入。

多项式回归模型的方程为
$$
y’ = b + \sum_i w_i x_i
$$
符号含义相似。


损失函数选择平方损失函数（也称为 L2 损失函数），表示为
$$
Loss = \sum_{(x,y) \in D} (y - y’)^2
$$
其中：

  • $(x,y) \in D$ 是包含多个有标签样本的数据集，这些样本为 $(x,y)$ 对。

一般的，有四种主要的损失函数，如下表所示：

损失类型 定义 公式
L1 损失 预测值与实际值之间差异的绝对值的总和。 $\sum
平均绝对误差 (MAE) 一组示例的 L1 损失的平均值。 $\frac{1}{N} \sum
L2 损失 预测值与实际值之间的平方差的总和。 $\sum(y − y’)^2$
均方误差 (MSE) 一组示例的 L2 损失的平均值。 $\frac{1}{N} \sum(y − y’)^2$

其中，$y$ 为真实数据（监督学习中的训练集数据）， $y’$ 为模型预测值

L1 损失函数和 L2 损失函数（或 MAE 和 MSE）之间的功能差异在于平方。当预测值与标签之间的差异较大时，平方会使损失变得更大。当差异很小（小于 1）时，平方会使损失更小。


逻辑回归

分类

逻辑回归的方程为
$$
y’ = \frac{1}{1 + e^{-z}}
$$
其中：

  • z 是线性方程的输出

逻辑回归模型的线性组件表示为：
$$
z = b + \sum_i w_i x_i
$$
其中：

  • z 是线性方程的输出（也称为 对数几率。
  • b 是偏差。
  • w 的值是模型学习的权重。
  • x 的值是特定样本的特征值。

逻辑回归的损失函数是 对数损失函数。通过 对数损失方程式会返回变化幅度的对数， 而不只是从数据到预测的距离。对数损失的计算公式为 如下：
$$
Loss = \sum_{(x,y) \in D} -y\log(y’) - (1-y)\log(1-y’)
$$
其中：

  • $(x,y) \in D$ 是包含多个有标签样本的数据集，这些样本为 $(x,y)$ 对。
  • $y$ 是有标签样本中的标签。由于这是逻辑回归， $y$ 的每个值都必须为 0 或 1。
  • $y′$ 是模型的预测结果（介于 0 和 1 之间），给定集合 $x$ 中的功能。

补充

逻辑回归模型的本质可以概括为：

逻辑回归假设的是因变量的“对数几率（log odds）”与自变量之间存在线性关系。

逻辑回归要解决什么问题？

逻辑回归主要用于分类问题，特别是二分类问题，比如预测某人是否患病、某封邮件是否是垃圾邮件等。

  • 因变量 $y$ 只能取两个值，通常设为 0 和 1。
  • 输入的是一个或多个自变量（特征） $x_1, x_2, \dots, x_n$。

我们希望建一个模型，输出：

$P(y = 1 \mid \mathbf{x})$

也就是，给定自变量 $\mathbf{x}$，预测 $y = 1$ 的概率。

为什么不能直接用线性模型？

最初的想法可能是用线性回归：

$P(y = 1 \mid \mathbf{x}) = \beta_0 + \beta_1 x_1 + \dots + \beta_n x_n$

但这不合理，因为：

  • 概率值必须在 $[0,1]$ 区间；
  • 而线性函数的值可以任意大或任意小。

所以我们需要一种变换，既能处理线性组合，又能保证输出落在 $[0,1]$ 区间。

引入几率（Odds）和对数几率（Log Odds）

  • 几率定义为：

    $\text{Odds} = \frac{P(y=1)}{P(y=0)} = \frac{p}{1-p}$

  • 对数几率（log odds）：

    $\log\left(\frac{p}{1-p}\right)$

对数几率是从概率空间映射到 $(-\infty, +\infty)$，适合做线性建模。

逻辑回归模型的核心假设

逻辑回归的核心假设是：

$\log\left(\frac{p}{1-p}\right) = \beta_0 + \beta_1 x_1 + \dots + \beta_n x_n$

也就是说：

自变量的线性组合影响的是“对数几率”，而不是概率本身。

通过变换得到 Sigmoid 函数（逻辑函数）

从上面的等式出发，解出 $p$（即 $P(y=1)$）：

$$\frac{p}{1-p} = e^{\beta_0 + \beta_1 x_1 + \dots + \beta_n x_n} \Rightarrow p = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x_1 + \dots + \beta_n x_n)}}$$

这个形式就是著名的 Sigmoid 函数：

$p = \sigma(z) = \frac{1}{1 + e^{-z}}, \quad z = \beta_0 + \beta_1 x_1 + \dots + \beta_n x_n$

逻辑回归不是对“概率”建模，而是对“对数几率”建模。

  • 本质假设：

    $\log\left(\frac{p}{1-p}\right) = \text{线性函数}$

  • 这样做的好处：

    • 保证概率值在 $[0,1]$
    • 模型保持线性模型的优雅（便于解释、训练）

K近邻

分类

k 近邻算法的输入为实例的特征向量，对应于特征空间的点；输出为实例的类别，可以取多类。

k 近邻算法假设给定一个训练数据集，其中的实例类别已定。分类时，对新的实例，根据其 k 个最近邻的训练实例的类别，通过多数表决等方式进行预测。因此，k近邻算法不具有显式的学习过程。

流程：

  1. 假设有一个带有标签的样本数据集（训练样本集），其中包含每条数据与所属分类的对应关系。
  2. 输入没有标签的新数据后，将新数据的每个特征与样本集中数据对应的特征进行比较。
    1. 计算新数据与样本数据集中每条数据的距离。
    2. 对求得的所有距离进行排序（从小到大，越小表示越相似）。
    3. 取前 k （k 一般小于等于 20 ）个样本数据对应的分类标签。
  3. 求 k 个数据中出现次数最多的分类标签作为新数据的分类。

决策树

分类或回归

决策树模型呈树形结构，在分类问题中，表示基于特征对实例进行分类的过程。它可以认为是 if-then 规则的集合，也可以认为是定义在特征空间与类空间上的条件概率分布。
决策树学习通常包括 3 个步骤: 特征选择、决策树的生成和决策树的修剪。

流程：

def createBranch():
检测数据集中的所有数据的分类标签是否相同:
If so return 类标签
Else:
寻找划分数据集的最好特征（划分之后信息熵最小，也就是信息增益最大的特征）
划分数据集
创建分支节点
for 每个划分的子集
调用函数 createBranch （创建分支的函数）并增加返回结果到分支节点中
return 分支节点


随机森林

分类

随机森林（Random Forest） 是集成学习方法的一种，它通过构建多个决策树并进行投票（分类）或平均（回归）来提升模型的准确率与鲁棒性。


支持向量机

分类

支持向量机是一种强大的分类和回归算法，核心思想是：寻找一个最优的超平面，将不同类别的样本最大间隔分开。

  • 对于线性可分问题，SVM 通过最大化边界间隔实现分类。
  • 对于线性不可分问题，SVM 使用核函数（如 RBF、Polynomial）将数据映射到高维空间，在高维空间中实现线性可分。
  • 支持向量 是位于分类边界上的样本点，对模型的决策起关键作用。

SVM 的损失函数通常是合页损失（Hinge Loss），其形式如下：

$$
Loss = \sum_{i=1}^N \max(0, 1 - y_i(w^T x_i + b))
$$

其中 $y_i$ 为真实标签（取值为 ±1），$x_i$ 为输入特征，$w$ 和 $b$ 为模型参数。


非监督学习

K-means 聚类

聚类

流程

  • 创建 k 个点作为起始质心（通常是随机选择）
  • 当任意一个点的簇分配结果发生改变时（不改变时算法结束）
    • 对数据集中的每个数据点
      • 对每个质心
        • 计算质心与数据点之间的距离
      • 将数据点分配到距其最近的簇
    • 对每一个簇, 计算簇中所有点的均值并将均值作为质心

降维

PCA

降维

流程

  1. 找出第一个主成分的方向，也就是数据 方差最大 的方向。
  2. 找出第二个主成分的方向，也就是数据 方差次大 的方向，并且该方向与第一个主成分方向 正交(orthogonal 如果是二维空间就叫垂直)。
  3. 通过这种方式计算出所有的主成分方向。
  4. 通过数据集的协方差矩阵及其特征值分析，我们就可以得到这些主成分的值。
  5. 一旦得到了协方差矩阵的特征值和特征向量，我们就可以保留最大的 N 个特征。这些特征向量也给出了 N 个最重要特征的真实结构，我们就可以通过将数据乘上这 N 个特征向量 从而将它转换到新的空间上。
