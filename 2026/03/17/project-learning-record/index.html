Project Learning Record

Zep

Graphiti 是一个用于构建和查询 AI 代理的时序上下文图的框架。与静态知识图谱不同，Graphiti 的上下文图跟踪事实随时间的变化，维护数据来源的溯源，并支持既定的和学习到的本体——这使得它们专为在不断发展、真实世界数据上运行的代理而设计。

与传统检索增强生成（RAG）方法不同，Graphiti 持续将用户交互、结构化与非结构化企业数据及外部信息整合为连贯、可查询的图。该框架支持增量数据更新、高效检索和精确历史查询，无需完整图重新计算，适用于开发交互式、上下文感知型 AI 应用。

Use Graphiti to: 使用 Graphiti 可以实现：

  • Build context graphs that evolve with every interaction — tracking what’s true now and what was true before.
    构建随每次交互而演变的上下文图——追踪当前真实情况及过往真实情况。
  • Give agents rich, structured context instead of flat document chunks or raw chat history.
    为智能体提供丰富、结构化的上下文，而不是平面的文档片段或原始的聊天历史。
  • Query across time, meaning, and relationships with hybrid retrieval (semantic + keyword + graph traversal).
    跨越时间、语义和关系进行查询，采用混合检索（语义+关键词+图遍历）。
