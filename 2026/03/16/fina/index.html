利润表 Income Statement

总收入 Total Revenue：直接收入

毛利率 Gross Profit：总收入 - 直接成本（含原材料费用，制造费用，人工成本等）

营业利润 Operating Income：总收入 - 直接成本 - 期间费用（员工工资、广告、公司场地的租金等）

能直接对应到每一杯奶茶/每一台手机/每一件衣服的员工工资 → 直接人工 → 影响毛利

不能直接对应到单个产品、属于公司整体运营支持的员工工资 → 期间费用工资 → 不影响毛利，只影响营业利润

净利润 Net Income：总收入 - 直接成本 - 运营费用 - 借钱利息 - 税款 - 其他开销

使用利润率 margin 进行行业内对比

例如，semiconductor gross margin 30%~70%


资产负债表 Balacen Sheet

总资产Total Assets：流动资产（银行存款、短期库存、1年期债券…） + 非流动资产（长期投资、机器、厂房…）

总负债Total Liabilities：流动负债（付给供应商的账款、短期内的借款、欠员工薪水…） + 非流动负债借款（一年以后需要还的长期债务，长期存款、长期债券）

总权益Total Equity：资产 - 负债 = 公司股本（发行股票时募集到的资金） + 留存收益（公司成立以来的积累利润 - 分红） + 库存股（回购股票）…


ROE 权益回报率：净利润 / 股东权益


现金流量 Cash Flow

现金正值流入公司，负值流出公司


经营现金流：= 净利润 + 非现金费用 +/- 营运资金变动

  • 净利润 (Net Income)：损益表上的最终利润，作为计算的起点。
  • 非现金费用 (Non-Cash Expenses)：需要加回（+）。因为计算利润时扣除了这些费用，但实际上并没有真实的现金流出。
    • 折旧与摊销 (D&A)：设备变旧了，但不用再掏钱。
    • 员工股权激励 (SBC)：给员工发股票，没发真金白银。
  • 营运资金变动 (Changes in Working Capital)：
    • 应收账款增加 -> 负值（-）：货卖了但钱没收回来，相当于资金被客户占用了。
    • 存货增加 -> 负值（-）：花钱进货/生产了，钱变成了仓库里的货。
    • 应付账款增加 -> 正值（+）：买了供应商的材料但还没付款，相当于无息占用了别人的钱。

投资现金流：=
[ - ] 资本支出 (CapEx) （买新机器、建新厂房流出的钱）
[ + ] 出售长期资产收到的现金 （把旧机器、旧厂房卖了换回来的钱）
[ - ] 并购与对外投资流出的现金 （花钱收购别的公司，或者买理财、买股票、买债券）
[ + ] 出售投资收到的现金 （卖掉理财、股票、债券，或者卖掉子公司换回来的钱）


融资现金流：借钱、发行股票（钱流入公司，正值；分红、回购股票（钱流出公司，负值


现金流净变动：= 营业现金流 + 投资现金流 + 融资现金流


自由现金流FCF Free Cash Flow：经营现金流 (CFO) - 资本支出 (CapEx)；公司在维持日常运营和必要扩张（资本支出）之后，真正可以自由支配的、能装进股东口袋里的真金白银。

DCF现金流折现估值

Discounted Cash Flow

公司未来产生的所有现金流折算到今天值多少钱

将公司未来的全部现金流，按照合适的折现率，折算成当前的价值

prc

  • 获取公司当前的自由现金流 -> search xxx free cash flow(FCF)

  • 预测公司未来n年的自由现金流 -> 过去n年向前推算 含主观预估

  • 选择合适的折现率 -> 加权平均资本成本WACC 计算折现率 含主观预估

  • 计算公司未来所有现金流的现值：=非永续增长阶段价值折现 + 永续增长阶段价值折现

非永续价值折现=未来n年的折现值的sum

永续价值折现 $V = \frac{FCF * (1 + g)}{WAAC - g} / (1 + WAAC)^{x}$，其中 $g$ 为永续增长率，FCF为非永续增长阶段最后一年的自由现金流，x为非永续年份

  • 计算公司的股权价值：= 非永续增长阶段 + 永续增长阶段 - 公司净负债net debt

  • 计算公司最终的合理股价：=股权价值/总流通股数outstanding shares


案例：

公司自由现金流 第0年 10000

预估未来增长率 15% ，折现率WACC 10%，$g$ 2%

未来第x年 $10000 * 1.15^{x}$

未来第x年到今年的折现现值 $10000 * 1.15^{x} / 1.1^{x}$

V = $\frac{10000 * 1.15^{x} * (1 + 0.02)}{0.1-0.02}$，永续折现值 $V/1.1^{x}$

六、常见问题排查

（一）数据接入失败

  1. 现象：输入分析指令后，系统提示“数据获取失败”或“API Key无效”；
  2. 排查步骤： 验证API Key正确性：重新检查配置的QVeris/Tushare密钥，确保无空格、换行； 测试网络连通性：

检查技能状态： openclaw skills list | grep qveris ，确保状态为“ready”； 重启服务： openclaw gateway restart 。

（二）定时任务未执行

  1. 现象：创建定时任务后，未按预期生成分析结果；
  2. 排查步骤： 检查Cron表达式格式：确保时间配置正确（如股市交易日为周一至周五）； 查看任务日志： openclaw logs --service cron --name pre-market-analysis 验证服务状态： openclaw gateway status ，确保服务正常运行。

（三）多平台联动无响应

  1. 现象：在飞书/Telegram发送指令后，机器人未回复；
  2. 排查步骤： 检查渠道配置： openclaw channels list ，确保状态为“connected”； 验证接口权限：确认飞书/Telegram机器人已开启“消息接收”权限； 查看渠道日志： openclaw logs --service channels --channel feishu

（四）数据更新延迟

  1. 现象：获取的股票数据与行情软件存在较大差异；
  2. 排查步骤： 更换服务器地域：选择中国香港/上海等靠近数据源的地域； 调整数据刷新频率：在指令中明确“获取实时数据”（如“查询A股实时涨幅榜，数据刷新频率1分钟”）； 切换数据接口：尝试用Tushare替代QVeris，对比数据差异。

七、安全与合规注意事项

  1. 密钥保护：QVeris/Tushare API Key是数据访问的核心凭证，切勿泄露给他人，定期更换密钥；
  2. 数据合规：仅用于个人投资分析，不得将数据用于商业用途或非法传播；
